Detailed Description

Class containing just one function, used to calculate change in species density.

The EcoChange class calculates change in species density taking the environmental constant and species optimum into account. The same formula as in StdChange is used (not inherited since it is very little code and we want to keep the calling stack to a minimum for performance).
